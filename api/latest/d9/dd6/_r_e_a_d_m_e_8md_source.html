13 **not_null** is a **0-overhead**, modern utility for ensuring non-nullability
14 in a coherent way.
15 
16 Unlike `gsl::not_null`, this type can work with move-only pointers like
17 `std::unique_ptr`, and does not require runtime checks unless explicitly
18 specified.
19 
20 ## Teaser
21 
22 ```cpp
23 auto register_widget(cpp::not_null<std::unique_ptr<Widget>> p) -> void
24 {
25  legacy_service.adopt_widget(std::move(p).as_nullable());
26 }
27 
28 ...
29 
30 // use 'cpp::check_not_null' to check and validate that 'p' is not null
31 register_widget(cpp::check_not_null(std::move(p)));
32 ```

36 ## Features
37 
38 * [x] Zero-Overhead; `not_null` has no hidden additional runtime
39  or storage overhead.
40 * [x] Written completely in <kbd>C++11</kbd>
41 * [x] Single-header, **header-only** solution -- easily drops into any project
42 * [x] No dependencies
